## 🧙‍♂️ Harry Potter Wiki

A React-based web application inspired by the Harry Potter universe.
The project combines an interactive quiz with a mini-wiki, allowing users to explore characters, spells, and test their knowledge.

## ✨ Features

- Browse characters and magical content
- Explore spells and the wizarding world
- Interactive quiz with video-based questions
- Track your results and progress
- Fast and modern UI built with Vite

## 🛠 Tech Stack

- React + TypeScript
- Vite
- React Router
- SCSS Modules
- ESLint + Prettier

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 📁 Project Structure

```
src/
  components/   // reusable UI components
  pages/        // app pages (routes)
  hooks/        // custom hooks
  types/        // TypeScript types
  styles/       // global styles
```

## 🎯 Purpose

This project was created to practice building modern React applications with a clean architecture, reusable components, and interactive UI logic.
